# MathGenius Uganda 🧮

A comprehensive mathematics education website designed specifically for Primary 6 and Primary 7 students following the Ugandan curriculum.

## 🌟 Features

- **2000+ Practice Questions** across all curriculum topics
- **Interactive Learning Interface** with real-time feedback
- **Multiple Practice Modes**: Quick practice, topic-specific, timed challenges, mock exams
- **Progress Tracking** with local storage and achievement system
- **Curriculum-Aligned Content** following Uganda National Curriculum Development Centre standards
- **Responsive Design** optimized for desktop, tablet, and mobile devices
- **Gamification Elements** including points, achievements, and streak counters

## 📚 Curriculum Coverage

### Primary 6 Topics
- Numbers & Operations (whole numbers, fractions, decimals)
- Measurement (length, area, volume, time, money)
- Geometry (2D shapes, angles, symmetry)
- Data Handling (charts, graphs, averages)
- Problem Solving (word problems, logical reasoning)
- Ratio & Proportion (basic ratios, proportional relationships)

### Primary 7 Topics  
- Advanced Numbers (large numbers, advanced fractions)
- Algebra Basics (simple equations, variables)
- 3D Geometry (solid shapes, surface area, volume)
- Advanced Statistics (probability, data analysis)
- Financial Mathematics (interest, profit/loss, budgeting)
- Advanced Problem Solving (multi-step problems, real-world applications)

## 🚀 Getting Started

1. **Open the website**: Simply open `index.html` in any modern web browser
2. **Navigate topics**: Use the topic tabs to explore Primary 6 and Primary 7 content
3. **Start practicing**: Click "Start Practice" to access the interactive practice interface
4. **Track progress**: Your scores and achievements are automatically saved locally
